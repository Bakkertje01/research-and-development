## Installation

### Windows
1. After downloading, unzip the archive, and move ipfs.exe somewhere in your %PATH%.
2. $ ipfs init
3. $ ipfs daemon
4. $ ipfs cat /ipfs/QmYwAPJzv5CZsnA625s3Xf2nemtYgPpHdWEz79ojWnPbdG/quick-start
5. $ open http://localhost:5001/webui
